Overview

Mission

Your team must boot a laptop using Kali Linux Live Mode and explore the cybersecurity tools inside the operating system.

Real cybersecurity analysts often spend time configuring systems and solving technical problems before they even begin security testing. This challenge simulates that experience.

⛔
Critical Rule — Do NOT Install Kali Linux.
You must select Kali Live only. Do not choose any install option.
◈
Downtown Donuts Context
You are acting as a security consultant for Downtown Donuts. Before any active testing begins, your team must verify that your tooling environment boots and operates correctly. This is standard pre-engagement procedure.

Part 1

Boot the System

Boot Procedure
  1. Insert the Kali USB drive into the laptop.
  2. Restart the laptop.
  3. While the system is starting up, press F12 to open the Boot Menu.
  4. Select the USB device from the boot list.
  5. Choose Kali Live — do not select any install option.
  6. Wait for Kali to finish loading before proceeding.
If It Doesn't Work — Troubleshooting Guide

Technical problems are expected. They are part of the challenge. Try to solve them using the guide below before asking for help.

Problem Possible Fix
Windows loads instead of Kali Restart → press F12 again → select USB boot from the menu.
USB device does not appear Restart → enter BIOS/Boot Menu → verify USB boot is enabled.
Secure Boot error Enter BIOS settings → locate Secure Boot → set to Disabled → save and restart.
Boot menu shows but Kali won't start Try a different USB port → restart and attempt again.
