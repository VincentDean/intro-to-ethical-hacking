// Mission Stakes

In this assignment, you have the ability to change your grade for this assignment.

Objective

Your objective is to locate and access a device on the network. Once you have gained SSH access, you will find a text document containing student grades. Using Nano, you will be able to view and modify those grade records.

What is a Brute Force Attack?
// Definition

A brute force attack is a method of gaining unauthorized access to a system by systematically trying every possible password or credential until the correct one is found. Rather than exploiting a technical vulnerability, it relies on persistence — and on the fact that many people use weak, predictable passwords. In this mission, you will use a list of the most commonly used passwords to attempt to authenticate via SSH.

Before You Start

// Network Scanning Primer

  • ifconfig Find your own IP address on the network.
  • nmap -sn <your-range>/24 Scan the network. Look for a device name that you think would have grades on it.

// Nano Primer

  • nano <filename> Open a file for editing.
  • Arrow keys Navigate within the file.
  • Ctrl + X → Y → Enter Save and exit.
// Critical A wrong move in Nano could corrupt the file. Make sure you understand how to save and exit before making any changes.
